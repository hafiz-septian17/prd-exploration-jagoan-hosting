# **App Name**: JUALIN: Sidoarjo UMKM Platform

## Core Features:

- Minisite Builder: Enable UMKM to build a simple minisite with a drag-and-drop editor for product display and branding.
- Product Catalog: Allow UMKM to list products with images, descriptions, and pricing.
- Training Module Access: Provide access to a training module with 7 sessions to help UMKM become professional entrepreneurs.
- Automated Promotion Tool: Generative AI powered tool to suggest promotional content (captions, hashtags) based on product details and chosen poster template. Can share content via WhatsApp and Instagram
- Performance Analytics: Track visits, popular products, and click-through rates to provide insights into business performance.
- Firestore Integration: Utilize Firestore for storing product listings, minisite data, and user analytics.
- User Authentication and Management: Secure user authentication and management for UMKM to control their storefront and data.

## Style Guidelines:

- Primary color: Soft Gold (#D4AF37) for a premium and elegant feel.
- Background color: White (#FFFFFF) to provide clean and spacious design.
- Accent color: Blue-Gold, an analogous hue to gold (#556B2F), with brightness and saturation adjusted for maximum contrast
- Body and headline font: 'Inter', a sans-serif font for a modern and clean look.
- Modern icons from Feather Icons or Phosphor Icons for a consistent and professional appearance.
- Clean spacing and soft rounded corners (10-16px) for a user-friendly interface.
- Smooth and subtle transitions for a seamless user experience. Micro interactions can be added for enhanced interactivity.
